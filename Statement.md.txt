Problem Statement
Managing day-to-day expenses is a challenge for many individuals, resulting in poor budgeting and lack of financial awareness. There is a need for a simple, smart solution to track personal spending accurately.

Scope of the Project
This project aims to provide a beginner-friendly Python application that allows users to record, categorize, and analyze daily expenses. The app offers visual summaries and spending insights, making expense tracking accessible for everyone.

Target Users
Students learning Python or personal finance

Individuals wanting to monitor expenses

Beginners interested in coding useful applications

High-Level Features
Add, edit, and categorize income and expenses

View summaries and graphical reports of expenditures

Get insights for better budgeting

Easy and intuitive user interface for quick adoption
